vscode 단축키

<매우 기본>
1. ctrl + /   (줄 주석)
2. shift + alt + a (block 주석)
3. ctrl + a   (전체선택)
4. ctrl + z   (코드 취소)
5. ctrl + x (한줄 잘라내기)
6. ctrl + f   (파일내 텍스트 검색)
7. ctrl + shift + f (프로젝트 폴더 전체 검색)
8. ctrl + ` (터미널 열기)


<기본>
1. ctrl + b  (사이브바 토글)
2. ctrl + y   (코드 복원)
3. ctrl + g :	행 이동
4. ctrl + p : 파일 이동
5. ctrl + enter (다음 행 삽입)
6. alt + 화살표 방향 (라인 이동)
7. alt + shift + 화살표 방향 (라인 복제)



<가끔 사용>
1. ctrl + d  (일치 단어 선택)
2. ctrl + shift + L (일치 단어 모두 선택)
3. alt + 마스 왼쪽 클릭 (부분 선택)
