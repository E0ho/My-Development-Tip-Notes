vscode 단축키

1. ctrl + /   (줄 주석)
2. ctrl + b  (사이브바 토글)
3. ctrl + a   (전체선택)
4. ctrl + y   (코드 복원)
5. ctrl + z   (코드 취소)
6. ctrl + f   (파일내 텍스트 검색)
7. ctrl + shift + f (프로젝트 폴더 전체 검색)
8. ctrl + d  (일치 단어 선택)
9. ctrl + shift + L (일치 단어 모두 선택)
10. ctrl + x (한줄 잘라내기)
11. alt + 마스 왼쪽 클릭 (부분 선택)
12. alt + 화살표 방향 (라인 이동)
13. alt + shift + 화살표 방향 (라인 복제)
14. ctrl + ` (터미널 열기)
15. ctrl + enter (다음 행 삽입)
